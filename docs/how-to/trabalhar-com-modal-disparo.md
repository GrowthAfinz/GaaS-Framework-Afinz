# Como trabalhar com o Modal de Disparo

1. O modal de disparo fica em `src/components/dispatch/`
2. O estado do formulário é gerenciado por `DispatchFormContext`
3. Cada seção é um componente Block separado
4. Projeções AI ficam nos componentes de `dispatch/ai/`
5. Inputs inteligentes usam `SmartSelect` e `SmartInput`
